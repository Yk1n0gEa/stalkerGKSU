# stalkerGKSU
It is a portal for finding github profiles of people related on the basis of organisations.

## Requirements

BeautifulSoup
<br>
`apt-get install python-bs4`
<br><br>
requests
<br>
`pip install requests`
<br><br>
Flask
<br>
`pip install Flask`
<br><br>
flask_table
<br>
`pip install flask_table`
<br>
## To run

Run the server app
<br>
`python routes.py`
